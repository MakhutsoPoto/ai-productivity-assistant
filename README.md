# AI_Productivity-Assistant
An AI-powered assistant that helps automate workplace tasks and boost productivity.
It is meant to intergrate functionalities meant to replace individual work tools and lessen repetitive daily tasks.

## Features
- Email generation
- Meeting summarization
- Task planning
- Research assistance
- Chatbot interaction

##AI functionality
  The application uses the Google Gemini API to power:
- Context-aware email generation
- Structured meeting summaries (key points, action items)
- Task breakdown and prioritization
- Research analysis with key insights 
- Conversational chatbot interaction
 Each feature uses tailored prompts to produce structured and relevant outputs

##  Technologies Used
- Lovable.AI
- Netlify
- Google Gemini API
- SupaBase


## Learning outcomes
- Apply AI tools effectively (e.g., ChatGPT, Gemini, Notion AI)
- Demonstrate strong prompt engineering skills
- Build a functional and practical solution
- Ensure responsible and ethical use of AI
